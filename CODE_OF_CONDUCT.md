# Code of Conduct

## Our Standards

We expect everyone interacting with this project — issues, PRs, discussions, code review — to:

- **Be respectful.** Disagree about ideas, not people.
- **Assume good faith.** Most miscommunication is just that.
- **Be constructive.** "This won't work because X" beats "this is bad".
- **Stay on topic.** This is a security tool, not a soapbox.

## Unacceptable Behaviour

- Harassment, personal attacks, or discriminatory language.
- Publishing others' private information without consent.
- Sustained disruption of discussions or reviews.
- Deliberate intimidation or trolling.

## Enforcement

Maintainers may, at their discretion, edit or remove comments, close issues or PRs, or block users who violate these standards. Persistent or severe violations result in a permanent ban.

## Reporting

Email **conduct@adjoint.uk**. Reports are confidential. We aim to respond within 72 hours.

## Scope

This applies to all project spaces — GitHub issues, PRs, discussions, commit messages, and any official communication channel.

---

This is a short, locally-maintained code of conduct rather than a copy of a longer standard text. The intent is the same: a project where contributors can do good work without being mistreated.
